## 8. 使用開始手順
1. `checklists/template.yaml`をコピーしてプロジェクト用チェックリストを作成
2. 必要に応じて項目をカスタマイズ
3. 開発時にチェックリストを参照
4. AIに「〜をチェックリストに追加して」と依頼して項目を追加
